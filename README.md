# LibreNMS-Install
This script is intended to automate the LibreNMS installation process according to https://docs.librenms.org/Installation/Install-LibreNMS/ using **Ubuntu 22.04** and **NGINX**. 

### How do I use it? 
On your designated server simply run the following commands: 
```
wget https://raw.githubusercontent.com/Navar0ne/LibreNMS-Install/master/install.sh
chmod +x install.sh
sudo ./install.sh
```
